How do I mark a project request as Supported, Not Supported, or Planned?

  • Open the designated project request record. Head over to the Ministry Review tab and there you will see the field "Ministry Review Status". Select the desired status and save the record.

How do I leave notes for the SD? There are certain project requests that require adjustments before they can be supported.

  • Open up the desired project request you wish to leave notes on. Select the timeline tab and create a note.
